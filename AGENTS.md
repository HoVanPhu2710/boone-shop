# AGENTS.md cho Boone Shop

## Tech Stack
- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS + shadcn/ui
- lucide-react cho icons
- Supabase (sẽ thêm sau)
- Stripe cho thanh toán

## Ngôn ngữ & Tiền tệ
- Toàn bộ website dùng tiếng Việt
- Đơn vị tiền tệ: VND (hiển thị dạng 299.000 ₫)

## Design Rules
- Luôn tuân thủ DESIGN.md (Shopify style) cho mọi component: màu sắc, typography, button, card, spacing, hover effect...
- Phong cách: Tối giản, hiện đại, trẻ trung, phù hợp áo thun unisex
- Mobile-first, responsive tốt

## Coding Rules
- Ưu tiên Server Components khi có thể
- Sử dụng shadcn/ui components
- Code sạch, có comment khi cần
- Tên route và component nên dùng tiếng Việt hoặc kebab-case rõ ràng

Khi tạo hoặc chỉnh sửa bất kỳ trang/component nào, trước tiên đọc DESIGN.md và AGENTS.md.
